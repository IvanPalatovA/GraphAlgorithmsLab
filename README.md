# GraphAlgorithmsLab  
Лабораторная работа №3 — Алгоритмы на графах (C++)

Проект реализует ориентированные и неориентированные взвешенные графы, алгоритмы поиска кратчайших путей, а также генерацию графов для проведения экспериментальных исследований.  
Дополнительно в проект встроена консольная игра **«Крестики-нолики»** как отдельный модуль.

---

## Реализованные задания
- **М-1.1** — Ориентированный граф (рейтинг 5)  
- **М-2.2.1** — Поиск кратчайших путей на ориентированном/неориентированном графе (рейтинг 5)  
- **М-3.1** — Генерация графов заданной топологии и размера (рейтинг 10)  

**Суммарный рейтинг:** 20 (при требуемом ≥ 12)

---

## Функциональность

### Работа с графами
- Представление ориентированных и неориентированных графов на основе списков смежности  
- Реализация алгоритмов:
  - Дейкстры  
  - Беллмана–Форда  
- Генерация случайных графов заданного размера и плотности  
- Сравнение времени работы алгоритмов на одинаковых входных данных  
- Экспорт результатов экспериментов в CSV  
- Консольный интерфейс с режимами:
  - ручного ввода  
  - автоматического тестирования  

### Дополнительный модуль
- Консольная игра **«Крестики-нолики»**
  - режим игры для двух игроков  
  - проверка победы и ничьей  
  - удобный текстовый интерфейс
